##Média de três provas

É pedido o valor de três provas e tirado a médias das três notas.

É feito o teste das médias com as seguintes exgências:

 * Se a média for menor que 3 o aluno está reprovado;

 * Se a média for maior ou igual a 3 e menor que 7 é mostrada uma mensagem dizendo    que o aluno fez a prova de recuperação e é solicitado a prova do exame.

   ( Se a nota do exame for menor que 5 o aluno está reprovado.

     Se a nota for maior ou igual a 5 o aluno é aprovado.)

* Se a média for maior ou igual a 7 o aluno está   aprovado.

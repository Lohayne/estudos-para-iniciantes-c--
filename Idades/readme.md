##Idades

Aqui é feita uma contagem de 10 duplas pedindo a seguinte informação:
- Idade da primeira pessoa;
- Idade da segunda pessoa.

É usado o laço de repetição while para fazer a contagem.

Dentro de cada laço são feito os testes com as seguintes idades:

 * Se a idade 1 for maior ou igual a 70 anos conta +1;

 * Se a idade 2 for maior ou igual a 70 anos conta +1;

 * Se a idade 1 for menor que 18 anos conta +1;

 * Se a idade 2 for menor que 18 anos conta +1;
 
 * Se a idade 1 for maior ou igual a idade 2 conta +1.

No final das 10 contagens é mostrado as seguintes informações:

- Quantidade de pessoas com mais de 70 anos;

- Quantidade de pessoas menores de idade;

- Quantidade de duplas em que as primeiras idades foram maiores que as segundas idades.

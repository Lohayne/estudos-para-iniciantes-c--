##Lanchonete

Sistema de uma lanchonete onde mostra o menu e é possível fazer o pedido de um produto do cardápio.

Primeiro é mostrado o Menu com os códigos de cada produto. 

Logo após surge uma mensagem pedindo para digitar o código do que deseja e que esteja no Menu.

Este código é testado pelo Switch case verificando qual foi o produto pedido e mostrando o seu preço.

Após a verificação, é pedido a quantidade deste produto escolhido e novamente é testado pelo Switch case para informar o valor a ser pago pelo pedido, da seguinte maneira:

 * Testa o código do produto e multiplica a quantidade pedida pelo valor do produto.

No final aparece o valor a ser pago e e uma mensagem agradecendo.
